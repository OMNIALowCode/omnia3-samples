### Customization
| Property | Description                     | Default Value |
|----------|---------------------------------|---------------|
| isReadOnly | Sets the elements as read-only (not allowing to add and remove images) |    true      |
| value | The current image: a string with TenantName/Entity Identifier/ File name |        |
