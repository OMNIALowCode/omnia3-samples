### Customization
| Property | Description                     | Default Value |
|----------|---------------------------------|---------------|
| identifier | Sets the text template name |          |
| parameters | Object representing text template parameters (see example below) | {}        |

**Example**

{
    
    parameter1: "parameter1 value",

    parameter2: "parameter2 value",

    ...

}
